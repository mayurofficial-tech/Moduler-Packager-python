# 🧰 Moduler & Packager - Multi Utility Toolkit

## 📌 Description

Moduler & Packager is a Python-based multi-utility toolkit that demonstrates modular programming by dividing functionalities into separate modules and integrating them into a single application.

This project combines multiple utilities such as date-time operations, mathematical calculations, random data generation, file handling, and module exploration.

---

## 🚀 Features

* 🕒 Date & Time Operations (Current time, difference, formatting, stopwatch, countdown)
* 🧮 Mathematical Operations (Factorial, compound interest, trigonometry, geometry)
* 🎲 Random Data Generator (Random numbers, list, password, OTP)
* 🔑 Unique Identifier Generator (UUID)
* 📁 File Operations (Create, write, read, append files)
* 🔍 Module Explorer (View attributes of any Python module)

---

## 🛠️ Technologies Used

* Python 3

  * Modules & Packages
  * File Handling
